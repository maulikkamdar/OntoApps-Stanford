An Ebola Virus-centered Knowledge Base

Broad Layout

The Ebola-KB Dashboard is an extremely interactive visualization dashboard that enables the user to explore all pertinent data and knowledge related to Ebola Virus and Ebola Virus Disease (as of May 2015), as well as providing advanced features to click and filter information, and links to external sources (PubMed, DrugBank, InterPro, Gene Ontology, and PDB).
The Ebola-KB Dashboard has the following widgets:

  1. Summarized List of the EBOV Genes and Protein Domains, and the EBOV Genomic Wheel,
  2. Publications associated with a genomic region and additional information for any selected publication,
  3. InterPro and Gene Ontology Annotations,
  4. MeSH Terms View,
  5. 3D-Molecular structure View for PDB entities,
  6. Associated Ligands and additional information on any selected ligand, and
  7. EBOV Genome Wheel that aids in the visualization of the spatial organization of the genomic regions.

The Widget A provides a summarized representation of the EBOV genes denoted using the consensus EBOV RefSeq entry, and the underlying protein domains listed in the Conserved Domain Database. The total number of publications and ligands associated with any particular genomic region are also shown.


Publications

The Widget B shows the list of PubMed publications that could be relevant to Ebola Virus and Ebola Virus Disease (isolated using genomic regions and E-Utils). We also offer provenance information by providing an external reference to the main source (blue box in the figure). Clicking on any particular publication entry shows additional information (article abstract, date, authors, journal, mesh terms) pertaining to that entity in an adjacent tab in Widget B (See the red box and diagram flow in the figure).

The Widget D (MeSH terms view) is interactive to the selection in Widget B. By default, all publications are shown as blue nodes in the force-directed layout, however, on selecting a specific publication, only those publications are shown which share MeSH terms (yellow nodes) with the selected publication. On hovering on the node, the shared MeSH terms are visible, as well as clicking on a node can also be used to navigate a certain publication. The Selected publication in Widget B is shown using a red-colored node, unless the selection is overridden by clicking on a genomic region (subsequently described).


Ligands

The Widget F show the list of all PDB ligands associated with any genomic region. Similar to Publications, we also offer provenance information by providing an external references to either PDB (share icon in the blue box) or DrugBank (leaf icon in the blue box in the figure). Clicking on any particular ligand shows additional information (molecular weight, SMILES, InChi Notation, etc.) pertaining to that ligand (see the red box and red diagram flow in the figure).

Clicking on any PDB structure link shows a interactive 3-dimensional structure view in Widget E. This structure view is generated using PDB SDF files. Clicking on any PubMed record link opens the publication in Widget B and also updates the MeSH terms viewer in Widget D.


Annotations

The Widget C show the list of all InterPro and Gene Ontology annotations associated with any genomic region. Similar to Publications, provenance information links out to InterPro or GeneOntology resources (share icon in the blue box). Clicking on any particular annotation shows additional relevant information in adjoining tab (see the red box and diagram flow).


Filtering

The Widget A allows you to filter publications, ligands, and annotations, by clicing either on the EBOV genes denoted using the consensus EBOV RefSeq entry, or the underlying protein domains listed in the Conserved Domain Database. As seen in figure, only those entities associated with the clicked gene or protein domain will then be shown.
